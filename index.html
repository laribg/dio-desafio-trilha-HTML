Breve resumo sobre o surgimento da Internet

(voltar)

A Internet nasceu de um projeto de pesquisa militar (ARPA: Advanced Research Projects Agency), no período da guerra fria. Inicialmente a ideia era conectar os mais importantes centros universitários de pesquisa americanos com o Pentágono para permitir a troca de informações rápidas e protegidas.

Com a ARPANet - Rede de Agencias de Projetos de Pesquisa Avançada ou Rede da Agência de Projetos de Pesquisa Avançada dos Estados Unidos, em 1969, surgiu a primeira conexão entre os computadores da Stanford e da UCLA.

Dois anos depois, o primeiro e-mail foi enviado por Ray Tomlinson. No início dos anos 70, nasceu a palavra Internet, que era aplicada ao sistema de redes interconectadas por meio dos protocolos TCP e IP (Transmission Control Protocol / Internet Protocol) nos quais eles baseavam serviços de Internet e e-mails.

Isso tudo até 1990, quando Tim Berners-Lee, conhecido como o pai da web, cria a linguagem HTML. A equipe do cientista criou o primeiro cliente web “World Wide Web” (www).


O que é um HTML? E o que são Tags?

(voltar)

O HTML é o componente básico de uma página na Internet, ele está presente em qualquer site, blog ou aplicativo da internet (inclusive nos apps dos nossos smartphones). A sigla HTML vem de Hyper Text Markup Language, cuja tradução literal seria Linguagem de Marcação de Hipertexto*, ou seja, o HTML não é uma linguagem de programação.

* Um hipertexto é um texto que possui links pelos quais o usuário pode navegar e escolher o seu próprio caminho de leitura.

O HTML é uma linguagem de marcação de hipertexto usada para marcar e dar significado às informações existentes em uma página. Assim sendo, a página de um site ou app contém marcações feitas com HTML. Essas marcações, chamadas de tags, são interpretadas pelo navegador. Este, por sua vez, “entende”, que determinado trecho da página contém um título, um parágrafo, uma imagem, uma lista, etc.

Uma tag HTML deve estar envolvida por um sinal de menor e um sinal de maior. Além disso, a maioria das tags HTML necessita de uma tag de abertura e outra de fechamento. Algumas tags, no entanto, não precisam de fechamento. Elas são chamadas de tags órfãs ou elementos vazios. Esse é o caso da tag de imagem img, por exemplo.

Uma tag HTML pode conter atributos (um, nenhum ou vários). Os atributos de uma tag HTML são elementos que acrescentam algum tipo de informação à tag. Por exemplo, a tag de imagem precisa, obrigatoriamente, conter o atributo src, que indica o caminho (o link) para a imagem em questão. Sem o atributo src e um link válido para um arquivo de imagem, nenhuma figura é exibida na página. Esses são alguns exemplos de tags que podemos adicionar ao HTML, abaixo segue a estrutura básica que um HTML deve conter para que o mesmo seja reconhecido pelo navegador como uma página web.
